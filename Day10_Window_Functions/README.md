# Day 10 – Window Functions in PySpark

This project demonstrates how to use PySpark window functions to rank
employees based on salary.

## Topics covered

- `Window`
- `partitionBy()`
- `orderBy()`
- `row_number()`
- `rank()`
- `dense_rank()`
- Finding the highest-paid employee in each department
- Finding the top two employees in each department
- Ranking employees across the complete dataset
